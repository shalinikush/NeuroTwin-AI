# 🧠 NeuroTwin-AI

## Overview
Brain tumor detection system using CNN and MRI scans.

## Features
- CNN-based detection
- High accuracy
- Dashboard visualization

## Tech Stack
- Python
- TensorFlow
- Streamlit / Flask

## Run

git clone https://github.com/shalinikush/NeuroTwin-AI.git
cd NeuroTwin-AI
pip install -r requirements.txt
python dashboard/app.py

## Note
Ensure model file exists in:
models/detection_model/brain_tumor_cnn.keras
